How It Works
The Edge Finder compares DraftKings lines against a composite probability model built from multiple independent sources. When the model's estimated probability exceeds the implied probability from the odds by a meaningful margin, an edge is identified.
Data Sources
DRatings provides power ratings and score predictions across major sports. Dimers supplies simulation-based win probabilities. ESPN contributes team stats, injury reports, and rest schedules. The Odds API provides real-time DraftKings odds for line comparison and player props.
Edge Calculation
Edge = Model Probability - Implied Probability. Picks must exceed a minimum edge threshold (3% for spreads/totals, 5% for props) to qualify. Higher-edge picks receive larger Kelly-criterion-based bet sizing, capped at 2% of bankroll per wager.
Bet Sizing
Wager amounts are calculated using a fractional Kelly Criterion approach. The formula accounts for edge size, odds, and current bankroll. Bets are capped to limit exposure and maintain bankroll longevity across variance-heavy stretches.
Confidence Levels
HIGH: Multiple sources agree, strong cushion beyond the spread, no schedule red flags.
MEDIUM: Sources mostly agree, moderate cushion, minor situational concerns.
LOW: Sources disagree (contested), thin margins, or back-to-back schedule impact.
